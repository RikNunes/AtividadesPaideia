Acessibilidade

A W3C tem vários padrões. Um deles é o WebDesign e Aplicações, Web Design e Aplicações referem-se aos padrões para o desenvolvimento de páginas Web, incluindo HTML5 CSS, SVG, Ajax, e outras tecnologias para Aplicações Web (“WebApps”). Esta seção inclui também insformações sobre como tornar páginas acessíveis para pessoas com deficiências (WCAG), sobre internacionalização, e ainda para dispositivos móveis. Para mais padrões acesse o W3C.

  1. Web para todos
  2. Web em todas as coisas
  3. Interação rica na Web
  4. Web de dados e serviços
  5. Web confiável
